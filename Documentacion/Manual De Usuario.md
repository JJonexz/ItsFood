Manual de Usuario  
Sistema Its Food

1. Introducción  
    Its Food es una aplicación web desarrollada con el objetivo de simular la gestión integral de un restaurante. El sistema permite visualizar y organizar información relevante del negocio, facilitando la comprensión de los distintos procesos operativos.
    

Está orientado tanto a usuarios que desean consultar el menú como a gestores que necesitan acceder a información administrativa.

2. Objetivo del sistema  
    El objetivo principal de Its Food es centralizar la información del restaurante en una sola plataforma. De esta manera se busca mejorar la organización, facilitar el acceso a los datos y establecer una base para futuras funcionalidades más avanzadas.
    
3. Requisitos del sistema  
    Para utilizar el sistema se necesita:
    

- Un navegador web actualizado (Google Chrome, Microsoft Edge u otro similar)
    
- No se requiere instalación adicional
    
- No es necesaria conexión a internet
    

El sistema funciona de forma local abriendo el archivo principal.

4. Acceso al sistema  
    Para acceder al sistema se debe:
    
5. Localizar el archivo index.html
    
6. Abrirlo con un navegador web
    
7. Navegar entre las secciones disponibles desde la página principal
    
8. Tipos de usuario
    

5.1 Vista de usuario  
Esta vista está destinada a clientes o usuarios generales.  
Permite:

- Visualizar el menú del restaurante
    
- Consultar productos disponibles
    

No permite realizar pedidos ni modificar información.

5.2 Vista administrativa  
Esta vista está dirigida a los gestores del restaurante.  
Permite:

- Acceder a información del sistema
    
- Consultar datos de distintas áreas del negocio
    

Actualmente no permite modificar ni guardar información.

6. Navegación general  
    El sistema está dividido en diferentes páginas, cada una con una función específica.  
    La navegación se realiza mediante enlaces que permiten acceder a:
    

- Inicio
    
- Empleados
    
- Ventas
    
- Pedidos
    
- Stock
    
- Caja
    

7. Uso de las funcionalidades
    

7.1 Visualización del menú  
Permite al usuario consultar los productos disponibles desde la página principal.

7.2 Gestión de empleados  
Muestra la información del personal del restaurante, incluyendo roles y organización.

7.3 Gestión de ventas  
Permite visualizar registros de ventas realizadas dentro del sistema.

7.4 Gestión de pedidos  
Presenta información sobre los pedidos y su estado.

7.5 Control de stock  
Permite consultar la disponibilidad de productos en el inventario.

7.6 Gestión de caja  
Muestra los movimientos de dinero, incluyendo ingresos y egresos.

7.7 Sistema de búsqueda e historial  
Permite buscar información dentro del sistema y consultar registros previos.  
Esta funcionalidad está implementada en el archivo shared.js.

8. Limitaciones actuales  
    El sistema presenta las siguientes limitaciones:
    

- No permite registrar nuevos datos
    
- No permite modificar información
    
- No almacena datos de forma permanente
    
- No cuenta con sistema de inicio de sesión
    

9. Mejoras futuras  
    Se proyectan las siguientes mejoras:
    

- Integración con base de datos
    
- Implementación de sistema de usuarios
    
- Registro real de pedidos y ventas
    
- Persistencia de datos
    
- Mejora en la interfaz y experiencia de usuario
    

10. Conclusión  
    Its Food es una aplicación en desarrollo que permite comprender el funcionamiento de un sistema de gestión para restaurantes.
    

Aunque actualmente se limita a la visualización de información, su estructura permite futuras ampliaciones para convertirlo en un sistema completo y funcional.
